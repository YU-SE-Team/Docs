# Software Requirements Specification (SRS)

| Project title       | 영남대학교 학생 식사 관리 서비스 |     |
| ------------------- | -------------------------------- | ----- |
| Project description | 본 프로젝트는 영남대학교의 학식 제도의 취약점을 보완하기 위하여 재학생이 의견을 나누며 안출한 방안을 토대로 기획되었다. 현재 영남대학교 교내 식당에서 식사를 하기 위해서는 식권 구매가 선행되어야 한다. 이 과정에서 특히 사람이 몰리는 점심시간에는 식당 바깥까지 줄이 이어지는 경우가 있으며, 데스크에 직원이 있음에도 식권 구매에만 상당한 시간을 할애해야 한다는 리스크가 존재한다. "Yummy Pass"는 이러한 대기 시간의 해소를 위하여 식권을 구매할 수 있는 사이트를 제공해 현장에서 대기할 필요 없이 바로 식권을 구매해 활용할 수 있는 서비스이다. |
