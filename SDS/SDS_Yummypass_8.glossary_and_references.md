# Glossary

| **용어** | **설명** |
| --- | --- |
| **Java Development Kit** | 약칭 JDK. Java 애플리케이션을 개발하기 위한 소프트웨어 개발 키트다. 이러한 개발 키트는 Java 프로그램의 컴파일, 실행, 디버깅에 필요한 도구들을 포함하고 있다. 개발자는 JDK를 활용하여 개발자는 플랫폼으로부터 독립적인 애플리케이션을 설계할 수 있다.  |
| **Spring Boot** | 웹 개발에 있어 복잡한 설정 없이 빠르게 필요한 라이브러리와 기타 환경을 설정하여 Spring 애플리케이션을 구축할 수 있도록 지원하는 프레임워크다.  Initializer에 접속해 생성할 프로젝트의 이름, 이용하고자 하는 의존성, 자동 빌드 도구를 선택하여 다운로드, IDE에서 실행시켜 실행하여 코드를 작성할 수 있다.  |
| **통합 개발 환경 (Integrated Development Environment)** | 약칭 IDE. 컴파일러, 코드 에디터, 그래픽 사용자 인터페이스, 디버거, 프로그램 에디터 등으로 이루어진 애플리케이션 패키지의 프로그래밍 환경이라고 할 수 있다. IDE는 자립형일 수도 있고 기존에 존재하는 특정 프로그램의 일부일 수도 있다.  |
| **Gradle** | Java, Kotlin 등 다양한 언어를 지워하는 오픈소스 빌드 자동화 도구이다. 의존성 관리에 용이하며 플러그인 시스템을 활용해 커스텀 플러그인을 작성할 수도 있다.  |
| **Lombok 어노테이션** | Java 개발 시 반복적으로 작성해야 하는 보일러플레이트 코드를 어노테이션을 통해 자동으로 생성해주는 라이브러리다. @Getter, @Setter, @ToString 등 다양한 옵션이 존재하며, 코드를 간결하고 유지보수를 용이하게 만든다. Entity, DTO 클래스를 작성하는 Spring Boot 프로젝트에서 자주 사용된다. |
| **QR코드** | Quick Response Code가 명칭인 2차원 바코드. 1차원 바코드보다 훨씬 많은 정보를 저장할 수 있는 매트릭스 형태이다. 흑백 사각형 패턴으로 구성되어 있으며, 스마트폰 카메라로 인식할 수 있다. 대용량 정보를 저장할 수 있으며, 인식 속도가 빠르다. 또한 어느 각도에서든 인식이 가능하도록 세 모서리에 위치 검출 패턴이 존재한다. QR코드는 ISO/IEC 18004로 국제 표준에 등록되어 있다. |
| **AWS** | Amazon 사에서 제공하는 클라우드 컴퓨팅 플랫폼이다. 트래픽 변화에 따라 자동으로 리소스를 확장하거나 축소할 수 있어 지출 비용을 효율적으로 관리할 수 있으며, 고가용성으로 전세계에서 다양한 리전과 다수의 가용 영역을 통해 안정적인 서비스를 제공한다. 주요 서비스에는 EC2와 같은 가상 서버 인스턴스를 제공하는 컴퓨팅 서비스가 존재한다. |
| **EC2** | AWS에서 제공하는 클라우드 컴퓨팅 서비스 중 하나이다. 가상 서버(인스턴스)를 클라우드 환경에서 생성하고 관리할 수 있으며, 사용자는 물리적인 서버를 구매하거나 유지보수할 필요 없이 필요한 만큼의 컴퓨팅 리소스를 확보할 수 있다는 장점이 존재한다.  |
| **Node.js** | Chrome V8 JavaScript 엔진을 기반으로 한다. 브라우저 밖에서 JavaScript를 실행할 수 있게 해주어 서버 사이드 애플리케이션 개발과 프론트엔드 개발 환경 구축에도 활용할 수 있다. 비동기 I/O 처리, 단일 스레드 이벤트 루프, 크로스 플랫폼 등의 특징이 존재한다.  |
| **React** | 메타에서 제공하는 프론트엔드 라이브러리. 싱글 페이지 애플리케이션이나 모바일 애플리케이션의 개발 시 토대로 사용할 수 있다. 현재 가장 활발하게 활용되는 웹/앱의 View를 개발할 수 있도록 하는 라이브러리이다. Data Flow, Component 기반 구조, Virtual Dom과 같은 특징을 지니고 있다. |
| **모달** | 모달은 대화창의 한 종류로 기본 창에 종속된 요소이다. 기본 창과 겹쳐져 가장 상단에 표시되며, 이때 기본 창은 비활성 상태로 전환되어 상호작용이 불가능하므로 사용자는 모달에서의 단일한 과업 또는 메시지에 집중할 수 있다. |
| **Wix** | Wix는 웹사이트 제작, 관리 및 성장을 위한 종합적인 도구를 제공하는 웹사이트 제작 플랫폼이다. 사용자는 처음부터 직접 웹사이트를 제작하거나 디자이너가 제작한 템플릿을 활용할 수 있으며, AI 웹사이트 빌더를 통해 자동으로 생성할 수도 있다. 드래그 앤 드롭 방식의 에디터를 통해 사이트의 세부 사항을 맞춤 설정할 수 있다. |
| **YP** | 서비스 “Yummy Pass”의 약칭이다.  |
| **식권** | “Yummy Pass” 서비스에서 구매한 학식 교환권을 가리키는 말이다. 식권은 서비스에서 결제한 이후 자동으로 생성되며, “My 식권”이라는 카테고리에서 확인할 수 있다. 매장에서 QR스캐너에 생성된 QR을 인식하면 식권이 사용 인증 되며 학식을 제공받을 수 있다. |
| **메인 페이지** | “Yummy Pass”를 사용하기 위해 접속했을 때 가장 먼저 볼 수 있는 페이지다. 로그인 컴포넌트가 존재하며, 해당 컴포넌트를 통해 로그인 또는 회원가입 페이지로 이동할 수 있다. 로그아웃하면 이 페이지로 리다이렉트된다. |
