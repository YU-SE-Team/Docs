| 17 | 매장 선택 |  |  |  |  |  |  |  |  |  |
|  |  |관리자가 관리할 매장을 선택할 수 있다.|  |  |  |  |  |  |  |  |
|  |  |  |해당 페이지로 접근하기 위해서는 관리자 로그인이 되어 있어야 한다.|  |  |  |  |  |  |  |
|  |  |  |해당 페이지에서 학생회관, 자연계, 교직원 중 하나를 선택할 수 있다.|  |  |  |  |  |  |  |
|  |  |  |매장을 선택하여 해당 매장의 메뉴들을 관리할 수 있는 페이지로 접근할 수 있다.|  |  |  |  |  |  |  |
| 18 | 현재 매장 메뉴 조회 |  |  |  |  |  |  |  |  |  |
|  |  |관리자가 현재 매장에 있는 메뉴들을 조회 할 수 있다.|  |  |  |  |  |  |  |  |
|  |  |  |매장을 선택하여 들어가면, 현재 매장에 등록되어 있는 메뉴들을 확인할 수 있다.|  |  |  |  |  |  |  |
|  |  |  |해당 화면에서 메뉴 등록, 추가, 수정이 가능하다.|  |  |  |  |  |  |  |
|  |  |  |메뉴 표시상태 토글을 통해 해당 메뉴를 사용자에게 표시할지 말지에 대한 설정이 가능하다.|  |  |  |  |  |  |  |
| 19 | 메뉴 등록 |  |  |  |  |  |  |  |  |  |
|  |  |관리자가 메뉴를 등록하고 가격, 남은 식권 수, 카테고리 등을 효율적으로 관리하도록 한다.|  |  |  |  |  |  |  |  |
|  |  |  |현재 매장의 메뉴 등록 버튼을 통해 추가가 가능하다.|  |  |  |  |  |  |  |
|  |  |  |팝업창을 통해 추가할 메뉴의 이름, 가격, 식권 매수, 카테고리, 이미지를 기입하고 등록이 가능하다.|  |  |  |  |  |  |  |  |필수 정보 중 하나라도 누락된 경우, 에러 메시지를 출력하고 생성을 거부한다.|  |  |  |  |  |  |  |  |
|  |  |  |등록이 성공하면 등록된 메뉴는 현재 매장 메뉴 조회 화면에서 등록 여부를 확인할 수 있다.|  |  |  |  |  |  |  |
|  |  |  |메뉴가 완전히 등록되기 전 팝업이 닫힌다면 적었던 내용은 저장되지 않는다.|  |  |  |  |  |  |  |
| 20 | 메뉴 수정 |  |  |  |  |  |  |  |  |  |
|  |  |관리자가 메뉴를 수정하고 가격, 남은 식권 수, 카테고리 등을 효율적으로 관리하도록 한다.|  |  |  |  |  |  |  |  |
|  |  |  |팝업창을 통해 수정할 메뉴의 이름, 가격, 식권 매수, 카테고리, 이미지를 변경하고 저장이 가능하다.|  |  |  |  |  |  |  |
|  |  |  |수정이 성공하면 수정된 메뉴는 현재 매장 메뉴 화면에서 반영된 상태를 확인할 수 있다.|  |  |  |  |  |  |  |
|  |  |  |메뉴가 완전히 저장되기 전 팝업이 닫히면 변경 내용은 저장되지 않는다.|  |  |  |  |  |  |  |
| 21 | 메뉴 삭제 |  |  |  |  |  |  |  |  |  |
|  |  |관리자가 메뉴를 삭제하여 더 이상 판매하지 않는 메뉴를 목록에서 지운다|  |  |  |  |  |  |  |  |
|  |  |  |현재 매장의 메뉴 삭제 버튼을 통해 삭제할 메뉴를 선택할 수 있다.|  |  |  |  |  |  |  |
|  |  |  |삭제가 성공하면 해당 메뉴는 현재 매장 메뉴 화면에서 목록에서 제거된다.|  |  |  |  |  |  |  |
| 22 | 식권 QR 사용 처리 |  |  |  |  |  |  |  |  |  |
|  |  |사용자가 보여주는 식권의 QR 코드를 인식하고 승인 절차에 활용한다.|  |  |  |  |  |  |  |  |
|  |  |  |식권 사용 처리를 위해 관리자는 QR 코드 인식 화면에 접속해야 한다.|  |  |  |  |  |  |  |
|  |  |  |사용자가 제시한 식권 QR 코드를 스캔하면 시스템에서 해당 식권의 유효성을 확인한다.|  |  |  |  |  |  |  |
|  |  |  |QR 코드가 유효하다면 해당 식권은 사용 처리된다.|  |  |  |  |  |  |  |
|  |  |  |사용 처리 결과는 사용자 화면과 관리자 화면에 반영된다.|  |  |  |  |  |  |  |
|  |  |  |만약 QR 코드가 유효하지 않다면 에러 메시지가 출력되고 사용 처리되지 않는다.|  |  |  |  |  |  |  |
